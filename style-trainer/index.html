🎯 이 도구의 목적

Radiant Blue Dot 웹사이트의 글을 AI가 써줄 때 "내 말투와 서술 방식을 모르면 소용없쟖아요".
그래서 이 도구에 당신만의 글쓰기 방식을 학습시키면, 나중에 AI에게
화성 소개 글 써줘 라고만 해도 내 스타일로 써줘요.

📋 규칙 타이핑 — 이렇게 써보세요
예시 1) 상황: 티저 문구 쓸 때
규칙: 첫 문장은 독자가 몰랐던 놀라운 사실로 시작해. 교과서 말투 절대 금지. '학교에서 배운 것처럼 딱딱하게'가 아니라 친구한테 이야기해주듯이.

예시 2) 상황: 전문 용어가 나올 때
규칙: 처음 나오는 전문 용어는 반드시 쉬운 비유나 일상 예시로 바로 설명해줘. '스펙트럼 = 무지개처럼 빛을 쪼개서 보는 것' 이런 식으로.

예시 3) 상황: 글 마무리할 때
규칙: 마지막 문장은 독자에게 질문을 던지거나 우주적 감동으로 끝내. 정보 나열로 끝내지 마.
📁 파일 업로드 — 뭘 올리면 좋을까요?
  • 직접 쓴 초안 — 메모장에 써둔 글도 OK
  • 좋아하는 책/글 발췌 — 이런 스타일이 좋다 싶은 텍스트
  • 윤하 가사, 칼 세이건 글 등 — 감성 레퍼런스
  • 웹사이트 티저 초안들 — 맛보기 텍스트 기존 버전
🚀 내보내기 활용법

1. "프롬프트 내보내기" 탭 클릭
2. "프롬프트 생성" 버튼 클릭
3. 생성된 텍스트 전체 복사
4. ChatGPT / Claude 새 채팅창 열기
5. 첫 번째 메시지로 붙여넣기
6. 그 다음부터 "화성 티저 써줘" 하면 내 스타일로 써줘요
