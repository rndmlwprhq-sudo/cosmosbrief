뉴스 소스 방식

현재 이 시스템은 Google News RSS 피드를 통해 각 기업/기관 관련 뉴스를 수집합니다.
RSS는 뉴스 배포를 목적으로 공개된 피드이므로 집계 자체는 합법적이에요.
단, 원문 텍스트를 그대로 복사하지 말고 직접 작성한 해설로 대체해주세요.

✅ 안전한 운영 원칙
  • 기사 원문을 그대로 쓰지 말고 편집장 해설 형식으로 작성
  • 수집된 기사 링크는 항상 출처 링크로 연결
  • 인용 시 최소한으로 (헤드라인 + 1~2 문장 이내)
  • 보도자료(Press Release)는 비교적 자유롭게 요약 가능
  • NASA, ESA 등 정부 공식 발표는 퍼블릭 도메인에 가까움
⚠️ 수익화 시 주의
  • 광고 수익 발생 시 → 요약문을 완전한 독자 저작물로 작성 필요
  • 구독료 모델 → 뉴스 API 유료 라이선스 검토 (NewsAPI Business: $449/월)
  • 수익이 월 $1,000 이상 → 법인 설립 + 저작권 정책 문서화 권장
