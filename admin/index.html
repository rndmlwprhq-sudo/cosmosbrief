🤖 OpenAI API 키

AI 초안 생성에 GPT-4o-mini를 사용합니다. platform.openai.com에서 발급 후 Vercel 환경변수로 등록하세요.
⚠️ 보안상 API 키는 Vercel 환경변수(OPENAI_API_KEY)에서만 관리합니다. 브라우저에 저장하지 마세요.

vercel.com → 프로젝트 → Settings → Environment Variables → OPENAI_API_KEY

⚠️ GITHUB_PAT / ADMIN_SECRET 설정 확인

Vercel 프로젝트 설정에서 아래 환경변수 2개가 모두 등록되어 있어야 합니다.
GITHUB_PAT — 기사 저장 (repo scope)
ADMIN_SECRET — 관리자 비밀번호 (무작위 문자열 권장)
vercel.com → 프로젝트 → Settings → Environment Variables

📦 이전 데이터 마이그레이션
